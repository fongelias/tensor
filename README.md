# Viktor
Javascript Machine Learning Library.

[![Build Status](https://travis-ci.org/fongelias/viktor.svg?branch=master)](https://travis-ci.org/fongelias/viktor)

## Installation
Viktor is available as an NPM module
```
npm install --save viktor
```
